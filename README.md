# Boomershub
React, NodeJS, ExpressJS, Sequelize, MySQL, Web Scraping

## Instructions to run:

Go to BoomershubFrontend, Open Terminal:
<p>npm install</p>
<p>npm run start</p>

Go to BoomershubBackend, Open Terminal:
<p>npm install</p>
<p>npm run start</p>

Open Browser, go to "localhost:3001"
